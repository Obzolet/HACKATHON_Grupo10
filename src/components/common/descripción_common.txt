Aquí tienen cabida, por ejemplo, los componentes para representar inputs de formularios, pintar imágenes de distintas formas, campos especiales como dropzones o elementos de UI como ventanas modales o sliders. Digamos que esta carpeta es el UI Kit de la aplicación y no posee ningún tipo de lógica, por lo que es habitual que recicle muchos componentes que aquí aparecen en distintos proyectos.
